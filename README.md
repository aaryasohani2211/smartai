# 🚀 Arya Sohani | AI/ML Engineer Portfolio

A premium, production-ready portfolio website built for a futuristic AI/ML showcase. Featuring high-end glassmorphism, 3D interactive visuals, and smooth inertial scrolling.

## ✨ Features

- **Futuristic AI Hero**: 3D mouse-reactive AI orb with floating tech stack pills and neural network animations.
- **Interactive About Section**: Glassmorphism competency cards with hover-triggered elevation.
- **Professional Journey**: Animated vertical timeline for experience tracking.
- **Bento Grid Projects**: Spotlight hover effects, gradient border glows, and interactive action overlays.
- **Skills Matrix**: Categorized technical skills with animated progress indicators.
- **Certification Gallery**: Diagonal light sweep effects on glassmorphism certificates.
- **Achievements Bento**: Animated number counters for competitive rankings.
- **Premium UX**: 
  - **Lenis Smooth Scroll**: High-performance inertial scrolling.
  - **Framer Motion**: Custom cubic-bezier reveal animations.
  - **Mouse Follower**: Interactive spring-animated cursor ring.
  - **AI Preloader**: Thematic loading sequence.

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) + [GSAP](https://greensock.com/gsap/)
- **Scrolling**: [Lenis](https://github.com/darkroomengineering/lenis)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

The project follows a **"Dark Futuristic AI"** theme:
- **Primary Background**: `#081120` (Deep Space)
- **Accents**: `#7C3AED` (Cyber Purple) & `#00D4FF` (Neon Cyan)
- **Typography**: Plus Jakarta Sans (Modern Geometric)
- **Effects**: Heavy use of `backdrop-blur`, radial glows, and animated mesh gradients to create a state-of-the-art SaaS feel.
